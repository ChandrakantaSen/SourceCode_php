Visitor Details tab

After login, Admin can view visitors’ details from visitor details tab. Admin is able to make visitors’ entry from here with a selected room type, check in and checkout date.Editing and Cancel of visitors’ records can also be done here.

To make New visitor entries, admin must follow the below mentioned steps-

Step1:Click on visitor entry and enter the details, i.e. room type, check in and checkout date .


Step2:After clicking the proceed button, it will show the available rooms as per the preference and admin will book one of those rooms.



Step3:After clicking on book option Admin will enter the necessary details. Please note, in total days section admin must enter the exact duration of accommodation or can add one more to the exact duration. Though, admin can’t give inappropriate duration, i.e., less than exact duration or more than exact duration+1. Whenever an inappropriate entry will be made, it will prompt the user with a dialog box to correct the entries.



For example, we made an entry with check in date on 2nd february,2016 and checkout date on 4th february,2016 and for this, duration or the total days will be entitled with 2days or 2+1= 3 days but not less than 2 days or more than 3 days.



Step4:After giving all correct details, the website calculates total rate automatically and allows room booking with book tab



Step5:After successful booking, website will show the entries in visitor details tab.



After successful booking, admin can edit and update necessary details or can cancel the booking details.


Admin can edit and update necessary details of Visitors' booking details.


Admin can cancel the booking. After cancellation of booking, cancel status will appear. Even, the booked room on these days will become available.

Visitors will receive a autogenerated confirmation mail and admin will address them as per their queries, within 3-4 hours of query submission.
